# AI-Based Predictive Maintenance System

## Overview
This project simulates a predictive maintenance system using sensor data and anomaly detection techniques.

The goal is to identify abnormal behavior in machine sensor readings that may indicate potential failures.

---

## Features
- Synthetic sensor data generation
- Anomaly detection using statistical thresholding
- Visualization of anomalies using matplotlib
- Modular project structure

---

## Tech Stack
- Python
- NumPy
- Pandas
- Matplotlib

---

## Project Structure

```
src/
├── main.py
├── anomaly_detection.py
```


---

## How it works
1. Generate simulated sensor data  
2. Introduce artificial anomalies  
3. Detect anomalies using statistical thresholds  
4. Visualize results  

---

## Example Output
The system detects anomalies in sensor data and highlights them in red on the graph.

- Blue line → normal sensor readings  
- Red points → detected anomalies  


---

## How to Run

1. Install dependencies:

pip install numpy pandas matplotlib

2. Run the script:

python src/main.py
